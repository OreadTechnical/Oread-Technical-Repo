# Oread Technical

Professional website for [Oread Technical](https://www.oreadtechnical.com) — aerospace stress engineering consulting by Jeremy deNoyelles.

## Site structure

- `index.html` — main landing page (services, experience, contact)
- `assets/Jeremy_deNoyelles_Resume_2026.docx` — downloadable résumé
- `css/styles.css` — styles
- `js/main.js` — mobile navigation
- `CNAME` — custom domain for GitHub Pages

## Local preview

Open `index.html` in a browser, or run a simple server:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080
